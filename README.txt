# OutfitBag

## Description
This FiveM script adds functionality for a saved outfit bag. Players can use the bag to view a list of previously saved outfits and quickly choose the desired one. It's particularly useful for players who want to change outfits frequently without having to manually save and load them.

## Features
- **View Outfit List:** Players can use the bag to view a list of previously saved outfits.
- **Outfit Selection:** Players can quickly select one of the outfits from the list to wear.
- **Intuitive Interface:** The user interface is simple and intuitive, allowing players to use the bag easily.

## Usage
1. Ensure that players have the `outfit_bag` item in their inventory.
2. Players can use the `outfit_bag` item from their inventory to open the outfit bag.
3. Inside the bag, players can view the list of previously saved outfits and select the desired one to wear.
